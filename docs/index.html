MycoReator

Objective

Control and monitor a heating pad system using Arduino and SHT30 sensor. Logs are visualized interactively. GitHub Repo

Capabilities

Logs temperature and humidity every 10 seconds. Controls heater via MOSFET. Generates interactive Plotly plots and Tabulator tables with download buttons. Highlights heater ON periods and shows target temperature.

Tasks

  • Read SHT30 temperature/humidity from Arduino
  • Control heating pad via IRL44N MOSFET
  • Log CSV data every 10 seconds in '../../results/heater'
  • Generate interactive plots and tables using Plotly.js and Tabulator
  • Wiring diagram: 
    SHT30 (I2C): Red->5V, Black->GND, Yellow->SCL (Mega21), White->SDA(Mega20)
MOSFET IRL44N: Gate->Pin9, Drain->Pad-, Source->GND, Pad+->12V+, Diode 1N4007 Cathode+12V, Anode Drain-
